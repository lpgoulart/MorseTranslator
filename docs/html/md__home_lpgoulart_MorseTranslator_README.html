MorseTranslator

Translator Morse <-> Alphabet

IMPORTANTE Os passos "2" em diante funcionam apenas em sistema operacional Ubuntu e variantes. **

Siga os passos abaixo para executar o programa:

  1. Para adquirir os arquivos insira no cmd de sua máquina:
1 git clone http://projetos.imd.ufrn.br/goulart/MorseTranslator.git
  1. Agora que você possui os arquivos vá para a pasta usando o código:
1 cd MorseTranslator/
  1. Em seguida, já dentro da pasta Morse Translator, digite:
1 make
  1. Este comando irá criar o diretório ../bin( onde o usuário encontrará o arquivo executável do programa ). Logo após gerar o diretório e o arquivo executável digite:
1 cd bin/
  1. Uma vez dentro do diretório ../bin insira para rodar o programa:
1 ./morse

Assim que dado o Enter, o programa estará rodando devidamente, para sair basta apenas digitar qualquer tecla para encerrar o programa.
